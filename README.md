# Daily Trading Strategy Research Platform

실시간 크립토 데일리 트레이딩 연구 플랫폼: 전략·시장·트레이더 스타일을 같은 규격으로 비교하는 시스템.

## 주요 기능
- **데이터 계층**: Binance Spot/Futures 및 Upbit WebSocket을 통한 실시간 데이터 수집 (체결, 호가, 스프레드 등)
- **전략 엔진**: 모멘텀 브레이크아웃, 평균회귀 스캘핑, 리드-래그 이벤트, 오더플로우 반응형 전략 등 다수 탑재 및 동일 인터페이스 구현
- **백테스트 및 평가**: 승률, Profit Factor, MDD, 시간대별 성과, 슬리피지 반영 손익 비교
- **트레이더 스타일 분석**: Jesse Livermore(추세 돌파), Linda Raschke(단기 셋업), Al Brooks(가격행동) 등 유명 트레이더 스타일을 메타데이터로 로드해 전략 비교
- **리스크 관리**: 일일 손실 제한, 연속 손절 중단, 주문 실패 중단 등 엄격한 리스크 제어

## 시작하기
1. 데이터 수집 봇 실행
2. 전략 백테스트 및 지표 추출
3. 실시간 대시보드를 통한 관측
